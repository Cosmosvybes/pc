# ParentingCertainty

An AI-powered "Psychological First Aid" application for parents. This PWA provides instant, clinically-grounded scripts for crises and regulatory activities for prevention.

## Features
### Crisis Mode
- **Instant Scripts**: Choose an Age + Situation (e.g., "6yo Bedtime") to get an exact "Say/Do/Avoid" script.
- **AI Intelligence**: Uses Google Gemini to dynamically generate unique scripts for complex situations.
- **Voice Guidance**: Text-to-Speech (TTS) reads the script calmly to the parent (or child).
- **Audio Regulation**: Generative "Brown Noise" (Rain/Storm) powered by Web Audio API for offline calming.

### Prevention Mode
- **Daily Wisdom**: Daily parenting stoicism/tips.
- **Concept Library**: Deep dives into "Connection Before Correction", "The 90-Second Rule", etc.
- **User History**: Tracks crisis frequency and situations handled.

###  Activity Finder (AI)
- **Occupational Therapy Engine**: Prompts the user for Energy Level + Time.
- **AI Generation**: Suggests *unique* regulatory activities using household items (e.g., "Pillow Sumo", "Ice Rescue").
- **Offline Fallback**: Robust local database of activities if internet is down.
